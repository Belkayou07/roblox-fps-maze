# Roblox FPS Maze

Minimal Roblox project foundation using Rojo.

## Current state

This repository only contains the basic project structure. No gameplay systems have been added yet.

## Structure

- `src/client` — LocalScripts and client-only code
- `src/server` — server Scripts
- `src/shared` — shared ModuleScripts
- `default.project.json` — Rojo project mapping

## Syncing

1. Clone the repository.
2. Install Rojo.
3. Run `rojo serve` from the repository folder.
4. Connect using the Rojo plugin in Roblox Studio.
